# PO Website uit Fundament 
Startcode voor het maken van de PO website uit\
[Fundament > Kernprogramma > A: Vaardigheden > 2. HTML & CSS (2025) > 5. Eindopdracht](https://fundament-online.nl/leeromgeving/content.php?id=58132)

Deze praktische opdracht maak je als volgt:
1. Je zorgt dat iedereen uit je groepje evenveel code maakt en je bewaart je werk elke les in GitHub.
2. Doe stap 1 tot en met stap 5 uit deze README. Zo leer je werken met de ontwikkelomgeving Codespaces.
3. Lees _paragraaf 5.1. Inleiding_ uit Fundament
4. In _paragraaf 5.2. Voorbereiding_ start je met coderen.\
   Gebruiken deze startcode in de ontwikkelomgeving Codespaces.\
   Maak geen map _eindopdracht_, maar zet je bestanden in de hoofdmap van je Codespace.\
   Doe verder wat aangegeven staat in paragraaf 5.2. 
6. In _paragraaf 5.3_ tot en met _paragraaf 5.6_ werk je aan de basis van je website.\
   Je maakt alle opdrachten in deze paragrafen.
8. Daarna maak je uitbreidingen aan je website die je zelf verzint.\
   Je mag het soort eten dat je verkoopt aanpassen, maar de meeste punten verdien je natuurlijk met nette html en css waarvan je zelf hebt uitgezocht hoe het werkt.\
   Zet een lijst van alle uitbreidingen die je hebt gemaakt in het bestand `uitbreidingen.md`.
10. De inleverdatum staat in de planner. Je docent kijkt na wat er op de deadline in GitHub staat.

## 1. Run code

- Open een Codespace
- Start een webserver in de terminal van de Codespace met het commando 
    ```
    npx -y vite
    ```
- Als je het commando knipt en plakt, dan verschijnt er de eerste keer links bovenin je scherm een popup waarin je toestemming moet geven.
- Om het commando uit te voeren druk je op de [Enter]-toets.
- Een popup verschijnt met de vraag of je een browser wilt openen verschijnt rechtonder in je scherm. Klik op de groene knop "In Browser Openen".
- Je ziet de inhoud van het `index.html` bestand in het geopende browserwindow. Klik op de link van de eerste opdracht om het resultaat te zien.

## 2. Wijzig code

- Voeg in het bestand `index.html` een nieuwe alinea toe. Bijvoorbeeld `<p>Deze alinea is van Piet</p>`.

## 3. Run gewijzigde code

- De aanpassingen worden automatisch in het browser-window getoond    

## 4. Bewaar je wijziging in GitHub

- Klik op het icoontje met de twee streepjes en 3 bolletjes (1)
- Type bij "Message" de opdracht waaraan je gewerkt hebt en enkele woorden wat je gedaan hebt (2)
- Klik op "Commit & Sync" (3)
